# Tanmayee Kodali - Personal Website

**Live site:** https://tanmayeekodali.github.io/my-website/

This is a personal academic and professional portfolio website built with R Markdown and deployed via GitHub Pages.

## What the Site Contains

### Home
A professional introduction including a bio, areas of expertise, recent academic and research highlights, and contact links.

### Projects
A collection of graduate-level and academic projects spanning:
- Climate adaptation and maternal health (cost-effectiveness analysis with interactive Shiny dashboard)
- Epidemiology and health informatics (PE-EHR+ study, SARS-CoV-2 variant analysis)
- Academic coursework projects in nutrition, program evaluation, health policy, and environmental health

### CV
A full curriculum vitae including education, work experience, publications, conference presentations, skills, awards, and leadership roles.

## Built With
- R Markdown website framework (`rmarkdown::render_site()`)
- Custom CSS (teal theme, card layouts, responsive design)
- Font Awesome icons
- Google Fonts (Merriweather + Work Sans)
- Deployed via GitHub Pages from the `/docs` folder
